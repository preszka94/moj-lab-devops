# DevOps Lab - Learning Repository

This repository is a learning space for DevOps, Python, and Bash development. It contains multiple projects organized by purpose.

## 📁 Structure

```
projects/
├── safedoc/          # Invoice processing system (Learning Project #1)
└── [new-projects]/   # Additional learning projects
```

## 🚀 Projects

### SafeDoc - Invoice Processing System

**Status**: Learning Project #1  
**Purpose**: Learn infrastructure automation, backend development, and data processing  
**Stack**: Python (Flask), Oracle Database, MinIO, Docker, Ansible, Google Cloud Build

See [projects/safedoc/README.md](projects/safedoc/README.md) for detailed documentation.

## 📚 Learning Goals

- **Python**: Backend development, data processing, automation
- **Bash**: Shell scripting, automation
- **DevOps**: Infrastructure as Code, containerization, CI/CD, configuration management
